### uv 项目管理
uv： Python 包 / 环境 / 版本三合一管理器
uv初始化项目cmd： uv init project-name
生成如下项目结构：
my-project/
├── .python-version   # uv 管理的 Python 版本文件
├── pyproject.toml    # uv 项目核心配置文件（依赖+元数据）
├── README.md         # 项目说明文档
└── main.py           # 你的主程序代码
